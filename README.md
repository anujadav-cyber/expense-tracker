# Expense Tracker 💰

A command-line expense tracker built with Python that helps you log, view, and analyze your spending.

## Features
- Add expenses with category, description, and amount
- View all expenses in a formatted table
- See spending summary grouped by category
- Data persists across sessions via CSV

## Tech Stack
- Python 3
- CSV module (built-in)
- datetime module (built-in)

## How to Run
```bash
python expense_tracker.py
```
